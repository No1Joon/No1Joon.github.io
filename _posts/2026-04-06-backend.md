## 왜 이 조합인가

<div class="callout why">
<div class="callout-title">FastAPI async</div>

GCP API 호출은 전형적인 **I/O 바운드** 작업이다. `async/await` 전략으로 스레드 블로킹 없이 동시 요청을 처리한다. WebSocket도 동일 서버에서 처리하므로 추가 인프라가 불필요하다.

```python
@app.websocket("/ws/live")
async def live_stream(ws: WebSocket):
    await ws.accept()
    async for event in collect_gcp_events():
        await ws.send_json(event)
```
</div>

<div class="callout why">
<div class="callout-title">SQLAlchemy 2.x async</div>

`async with session`, `select()` 스타일로 동기 코드와 완전히 분리된다. 1.x의 `session.query()` 패턴을 버리고 core-like API로 통일.

```python
async with AsyncSession(engine) as session:
    result = await session.execute(
        select(Scenario).where(Scenario.id == scenario_id)
    )
    return result.scalar_one_or_none()
```
</div>

<div class="callout why">
<div class="callout-title">Pydantic v2</div>

모델 정의와 직렬화 속도가 v1 대비 **5–10× 빠름**. `from_attributes = True` 설정으로 ORM 모델 → Response 스키마 변환이 자동화된다.

```python
class ScenarioResponse(BaseModel):
    model_config = ConfigDict(from_attributes=True)

    id: int
    name: str
    created_at: datetime
```
</div>

<div class="callout why">
<div class="callout-title">Fernet 암호화</div>

GCP Service Account JSON을 디스크에 저장할 때 **대칭 암호화** 적용. 복호화 후 임시 파일 생성, 사용 즉시 삭제하는 패턴으로 키 노출 시간 최소화.

```python
fernet = Fernet(settings.ENCRYPTION_KEY)
encrypted = fernet.encrypt(sa_json_bytes)
# DB 저장 후
decrypted = fernet.decrypt(stored_bytes)
```
</div>
