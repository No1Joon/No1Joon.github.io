## Flash 색상 — Google 브랜드 팔레트

<div class="callout">
<div class="callout-title">설계 원칙</div>

Flash 색상 4종은 **Google 브랜드 팔레트** 기반이다. GCP 프로젝트의 네트워크 이벤트를 시각화하는 맥락에서 사용자가 이미 친숙한 색상 체계를 활용했다.
</div>

노드 이벤트 유형과 1:1 매핑:

| Flash 색상 | Hex | 이벤트 유형 |
|-----------|-----|------------|
| Green | `#34A853` | 성공 (2xx, 정상 패킷) |
| Amber | `#FBBC04` | 경고 (지연, 재시도) |
| Red | `#EA4335` | 에러 (4xx, 5xx, 타임아웃) |
| Blue | `#4285F4` | 정보 (연결 수립, 헬스체크) |

## Canvas 배경 — GitHub Dark

`#0D1117`은 GitHub의 다크 모드 캔버스 색상과 동일하다. GCP 인프라 시각화를 "우주 공간에 떠 있는 노드" 이미지로 표현하기에 적합하다.

```css
:root {
  --canvas-bg: #0D1117;
  --panel-bg: #161B22;
  --border: #30363D;

  /* Flash 색상 */
  --flash-green: #34A853;
  --flash-amber: #FBBC04;
  --flash-red:   #EA4335;
  --flash-blue:  #4285F4;
}
```

## GSAP Flash 애니메이션

패킷이 노드에 도달하면 해당 이벤트 유형의 Flash 색상으로 노드를 순간적으로 빛나게 한다.

```ts
function flashNode(nodeRef: RefObject<SVGCircleElement>, eventType: EventType) {
  const color = {
    success: "#34A853",
    warning: "#FBBC04",
    error:   "#EA4335",
    info:    "#4285F4",
  }[eventType]

  gsap.timeline()
    .to(nodeRef.current, { fill: color, scale: 1.3, duration: 0.1, ease: "power2.out" })
    .to(nodeRef.current, { fill: "#4C4C4D", scale: 1.0, duration: 0.4, ease: "power2.in" })
}
```

## CSS 변수 적용 예시

```css
.node {
  fill: var(--primary);
  stroke: var(--border);
  transition: fill 0.1s;
}

.panel {
  background: var(--panel-bg);
  border: 1px solid var(--border);
  border-radius: 8px;
}

.event-badge[data-type="success"] { background: color-mix(in srgb, var(--flash-green) 15%, transparent); color: var(--flash-green); }
.event-badge[data-type="error"]   { background: color-mix(in srgb, var(--flash-red) 15%, transparent);   color: var(--flash-red); }
```
