## 병렬 수집 구조

Live 모드에서 4개 소스를 `asyncio.gather()`로 동시에 수집한 뒤 WebSocket으로 프론트엔드에 스트리밍한다.

```python
async def collect_all(project_id: str, credentials) -> LiveSnapshot:
    logging_data, monitoring_data, trace_data, compute_data = await asyncio.gather(
        fetch_vpc_flow_logs(project_id, credentials),
        fetch_metrics(project_id, credentials),
        fetch_trace_spans(project_id, credentials),
        fetch_instance_mapping(project_id, credentials),
    )
    return LiveSnapshot(
        logs=logging_data,
        metrics=monitoring_data,
        traces=trace_data,
        nodes=compute_data,
    )
```

순차 실행 대비 **최대 4× 빠른 응답**. 각 API의 지연 시간이 독립적이므로 병렬화 효과가 크다.

## 각 SDK 역할

### Cloud Logging — VPC Flow Log 수집

```python
from google.cloud import logging as gcp_logging

client = gcp_logging.Client(project=project_id, credentials=creds)
entries = client.list_entries(
    filter_='resource.type="gce_subnetwork"',
    order_by=gcp_logging.DESCENDING,
    max_results=500,
)
```

VPC 내 패킷 흐름(소스 IP, 대상 IP, 포트, 바이트 수)을 수집해 네트워크 토폴로지를 구성한다.

### Cloud Monitoring — 인스턴스 메트릭

```python
from google.cloud import monitoring_v3

client = monitoring_v3.MetricServiceClient(credentials=creds)
results = client.list_time_series(
    name=f"projects/{project_id}",
    filter='metric.type="compute.googleapis.com/instance/cpu/utilization"',
    interval=interval,
    view=monitoring_v3.ListTimeSeriesRequest.TimeSeriesView.FULL,
)
```

CPU, 네트워크 처리량 등 메트릭을 노드별로 수집해 시각화에 사용한다.

### Cloud Trace — 분산 트레이싱

서비스 간 요청 흐름을 스팬 단위로 수집. 특정 서비스의 지연 원인을 추적하는 데 활용한다.

### Cloud Compute — IP → NodeId 매핑

```python
from google.cloud import compute_v1

instances_client = compute_v1.InstancesClient(credentials=creds)
```

GCP 인스턴스의 내부 IP를 NodeId로 변환하는 매핑 테이블을 구성. Flow Log의 IP 주소를 시각화 노드와 연결하는 데 필요하다.

## 인증 처리

Service Account JSON을 Fernet으로 암호화해 DB에 저장 → 요청 시 복호화 → 임시 파일 생성 → SDK 초기화 → 임시 파일 삭제 순으로 처리한다.

```python
import tempfile
from cryptography.fernet import Fernet

def get_credentials(encrypted_sa_json: bytes, key: bytes):
    fernet = Fernet(key)
    sa_json = fernet.decrypt(encrypted_sa_json)

    with tempfile.NamedTemporaryFile(suffix=".json", delete=True) as f:
        f.write(sa_json)
        f.flush()
        return service_account.Credentials.from_service_account_file(f.name)
```
